## 2D 參考線畫布、透視線畫布

| 操作           | 指令                   |
| -------------- | ---------------------- |
| 安裝依賴       | `pnpm install`         |
| 啟動開發伺服器 | `pnpm dev`             |
| 打包建構       | `pnpm build`           |
| 預覽建構結果   | `pnpm preview`         |
| 安裝套件       | `pnpm add 套件名稱`    |
| 移除套件       | `pnpm remove 套件名稱` |
